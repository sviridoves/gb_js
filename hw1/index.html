Будет равно "1000108". Будет произведено преобразование числовой переменной в строку и конкатенация строк

4. * Самостоятельно разобраться с атрибутами тега script (async и defer).

Без данных атрибутов: в то время как браузер натыкается на тег script парсиг html документа ставится на паузу, производится загрузка скрипта и его выполнение. Парсин продолжается после завершения выполнения скрипта.

Если использовать атрибут async, то во время загрузки скрипта парсинг не останавливается. Остановка парсинга происходит только на момент выполнения скрипта.

Если использовать атрибут defer, то сначала производится полностью парсинг страницы, скрипт при этом может загружаться параллельно с парсингом. Выполнится скрипт только после окончания парсинга.
